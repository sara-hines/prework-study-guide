HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The document type declaration tells the browser what document type it should expect when it is opened, in this case, html.
  • The html opening tag represents the root, or start, of our HTML document.
  • An HTML attribute defines additional information about an element.
  • The attribute of lang="en" tells our browser the language of the webpage is English.
  • The title element defines the title for the webpage.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The meta element defines the metadata such as character set, description, keywords, author, and viewport.

CSS

  • CSS defines how webpage content is presented to users, including the colors, aesthetics, visual layout, and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git branch: lists your branches
  • git revert SHA-of-most-recent-version-before-bug: reverts to an earlier version of your code, before a bug took place, using the SHA of the appropriate version to return to
  • git clone SSH-link-of-repo-you-want-to-clone: clones a repo of your choosing to your current directory on your computer

JavaScript

  • JavaScript adds interactivity to websites; Using JavaScript, you can power your websites with effects, interactivity, data communication, and more.
